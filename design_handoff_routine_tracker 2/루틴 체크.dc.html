화면 플로우 — 탭바로 이어지는 4개 화면 + 편집 폼

하단 탭바(캘린더 · 오늘 · 통계 · 설정)로 이동합니다. 각 폰을 직접 눌러 탭을 바꾸고, 날짜 체크·루틴 추가/편집·표시숨기기까지 동작해요. 캔버스를 팬/줌하며 플로우를 한눈에 보세요.

1온보딩 · 빈 상태
루틴 0개일 때 첫 화면. '첫 루틴 만들기' → 추가 폼.
2오늘
오늘 할 루틴만 크게. 진행 링 + 큰 체크 토글.
3캘린더
주 단위 위클리 카드. 날짜 탭 → 체크 시트, 달성 주는 빛남.
4통계 대시보드
전체 화면 인사이트. 요약 카드 + 루틴별 달성률·연속·주간 히트맵.
5설정 · 루틴 관리
루틴 목록(표시숨기기·편집)·알림·주 시작요일. 루틴 탭 → 편집 폼.
6루틴 추가/편집 폼
전체 화면 폼. 이름·아이콘·색·목표(이상/이하) 편집.
